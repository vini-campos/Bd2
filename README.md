## 📁 Estrutura

```txt
/bd2
│
├── /procedures -> storaged procedures de Locação e clientes
│   ├── /triggers
│   │    └── trigger_filmes.sql
│   ├── procedure_ClientesCidade.sql
│   ├── procedure_DevolverFilme.sql
│   ├── procedure_incluirLocacao.sql
│   ├── procedure_aniversario.sql
│   └── procedure_crud.sql
│
└── /venda -> sistema financeiro, tabelas e inserts
    ├── /triggers
    │   └── triggers_venda.sql
    ├── inserts.sql
    └── venda.sql
```

## 🛠️ Tecnologias

- SQL Server
- T-SQL

## 📌 Destaques

- Triggers para automação de regras
- Modelagem de tabelas com relacionamentos
- Lógica de verificação de dias úteis para cobrança de parcelas
- Stored procedures organizadas por contexto
